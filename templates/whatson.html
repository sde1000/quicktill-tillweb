What's powering this?

The tills are run using free software.

  • The till software
  • This web service ("emf" branch)
  • The display boards fetch web pages like https://bar.emfcamp.org/display/on-tap.html

Pull requests welcome! If you improve the web service, your changes may be made live during EMF!

A simple JSON interface is available:

  • http://bar.emfcamp.org{% url "locations-json" %} lists the groups of stock that are on sale at the moment
  • http://bar.emfcamp.org/location/[name].json lists stock on sale in a location, for example the Bar location
  • http://bar.emfcamp.org{% url "stock-json" %} lists everything we have left in stock.
  • http://bar.emfcamp.org{% url "progress-json" %} compares how much we've actually drunk with what we predicted before the event...
